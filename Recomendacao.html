    Palestras e estudos

    Palestra Clean Code (Código Limpo)
    Clean Code é um termo da engenharia de software originado no livro de mesmo nome de Robert Cecil Martin . Os desenvolvedores de software descrevem principalmente o código-fonte como “limpo” , mas também documentos, conceitos, regras e procedimentos que são intuitivamente compreensíveis. Qualquer coisa que possa ser corretamente entendida com pouco esforço e em pouco tempo é considerada intuitivamente compreensível.

    Palestra - Estratégias de SEO para Desenvolvedores Front End
    Assuntos como SEO e acessibilidade são geralmente ignorados por desenvolvedores Front-end, por falta de demanda no projeto ou por não achar importante adicionar este tipo de tarefa em suas tarefas diárias. Nessa palestra vamos aprender algumas estratégias e técnicas para otimizar uma página Web, melhorando seu SEO e também sua acessibilidade.
